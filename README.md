# 🔥 FocusFlow

FocusFlow is a simple and effective productivity web app designed to help users stay focused, manage tasks, and track their progress using the Pomodoro technique.

---

## 🚀 Features

### ⏱️ Pomodoro Timer
- 25-minute focus timer
- Start, Pause, and Reset functionality
- Alerts when session ends

### 📝 Task Manager
- Add tasks easily
- Mark tasks as completed
- Delete tasks

### 🔥 Streak System
- Increases when tasks are completed
- Encourages daily consistency

### 📊 Productivity Score
- Score increases after completing a timer session
- Helps track productivity level

### ⌨️ Interactive Feature
- Press **Enter key** to quickly add tasks without clicking the button

---

## 💾 Data Storage

- Uses **LocalStorage**
- Saves:
  - Tasks
  - Streak
  - Score
- Data remains even after refreshing the page

## 📌 How to Use

2. Start the timer and focus on your task
3. Mark task as complete when done
4. Track your streak and score
